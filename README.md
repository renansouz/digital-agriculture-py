## 📜 Descrição

Este projeto foi desenvolvido pela equipe FarmTech Solutions com o objetivo de simular uma aplicação voltada à Agricultura Digital, utilizando Python e R para atender aos requisitos da atividade acadêmica.

Na etapa em Python, o sistema permite cadastrar, listar, atualizar e remover dados de duas culturas agrícolas, organizadas em vetores, além de calcular área de plantio e manejo de insumos. As culturas escolhidas para o projeto foram café arábica e tomate para processamento industrial, por representarem cenários agrícolas distintos e por permitirem a aplicação de fórmulas geométricas diferentes no cálculo da área.

A aplicação em Python também utiliza estruturas de repetição e decisão, além de um menu interativo para navegação entre as funções principais. O objetivo foi manter o código simples, funcional e fácil de apresentar, sem perder o alinhamento com o que foi solicitado no enunciado.

Na etapa em R, o projeto realiza cálculos estatísticos básicos, como média e desvio padrão, com base nos dados gerados na aplicação em Python. Além disso, foi adicionada uma integração com a API pública de clima Open-Meteo, permitindo a consulta e exibição de dados meteorológicos diretamente no terminal, como temperatura e velocidade do vento.

Na disciplina de Formação Social, o grupo realizou o resumo do artigo sobre o uso de veículos aéreos não tripulados (VANTs) na agricultura de precisão, seguindo a formatação exigida pela atividade.

O projeto foi organizado com versionamento no GitHub para simular um ambiente colaborativo de desenvolvimento, reunindo os arquivos de código, o resumo do artigo e o link do vídeo demonstrativo em um único pacote final ZIP.
